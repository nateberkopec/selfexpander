What is the Self-Expander?

The Self-Expander suggests a random Very Short Introduction (a book) or a Great Course (usually an audiobook).

Use the Self-Expander to randomly select a new thing to learn about. We suggest:

  • Study this subject for 1 month. This is the time it takes most people to read a Very Short Introduction or listen to a Great Course.
  • Push the button only once. Part of this experiment is to learn about things you may not have any interest in right now. Self-expansion happens when we learn how to appreciate something.
